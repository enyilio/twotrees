目前網站建置最常使用的主機為虛擬主機與VPS雲端主機，其主要原因為成本考量與穩定的效能。以下為虛擬主機與VPS雲端主機介紹，以及在什麼情況下，適合使用何種類型主機，會以「網站類型」以及「服務地區」兩個項目來評估並推薦網站主機商。

### 虛擬主機介紹(Shared Web Hosting)

虛擬主機是指在一台伺服器下，以共享主機硬體資源的概念，分割出多個虛擬伺服器主機服務。

##### 優點：

因為共享同一個主機資源，所以主機價格便宜實惠，同時都具有主機介面方便操作，即使不懂安裝主機環境或者原理，仍然可以透過標準化流程進行主機管理。					

##### 缺點：

因為共享主機資源，所以在主機的利用率上無法太高，無法承受較大的使用頻寬與流量，同時連線速度與雲端主機相比明顯較慢，有著明顯的連線速度差異。					

- 注意事項

慎選合作的虛擬主機服務商，因虛擬主機為共享主機資源，因此是跟許多人一起共用一台主機資源，包含共同的主機IP，若當某些用戶不當使用主機資源，例如濫發EDM或者網站被駭成為釣魚網站，都會影響主機IP信譽，一旦IP信譽受到影響，都會影響到網站在搜尋引擎上的排名以及郵件服務也有可能被屏遮。

### 虛擬主機適用時機

#### 適用網站類型

由於虛擬主機是共用主機資源，因此實際在硬體資源上會受到較多限制，比方說只能1小時發送300封信，或者無法同時間乘載大量訪客進入網站，因此如果是要放置形象網站、個人工作室，或者小型部落格，初期會比較建議先使用一般虛擬主機，因為價格較便宜實惠。

#### 建議服務地區

確定了網站類型，接著是網站主要服務的對象所在地為何？假若你的產品主要是外銷歐美，服務對象是歐美人士，那麼我們就會建議你使用歐美的主機來建置網站，如此對於歐美的客戶連線到歐美的主機會比網站放在台灣或亞洲地區的的主機，使用者體驗上的連線速度會有更明顯差異。

### VPS雲端主機介紹(Virtual private server)

VPS雲端主機是指一台主機伺服器下，分割多個專用伺服器，硬體資源CPU、記憶體、系統組態皆為獨立使用，獨立作業系統，同時配有獨立主機IP。

##### 優點：

更好的主機效能與連線速度，能乘載與因應瞬間流量湧入，更彈性化，可依實際需求進行主機環境參數配置。					

##### 缺點：

相較於虛擬主機費用高出許多，通常無主機管理介面，因此需自行安裝主機環境，需具備主機指令相關技能，對新手來說較困難上手。					

- 注意事項

VPS雲端主機可細分為VPS主機與雲端主機，其技術上是相同的，但不同之處在收費方式與主機規格調整方式。

VPS主機通常是提供不同的主機規格方案收費，可向上升級。而雲端主機則採用流量以及採用的規格計價，也就是用多少就收費多少，好處是可以隨時且更細緻調整主機規格因應不同情況的發生，但收費一般會比VPS主機固定規格來得高一些。

### VPS雲端主機適用時機

#### 適用網站類型

享有獨立主機資源，不會受到資源限制，同時可以隨時增加主機規格來因應短暫瞬間的流量進入，適用於流量與頻寬需求的高活動網站、電商平台、大型論壇以及媒體網站等。

#### 建議服務地區

與虛擬主機相同，取決在於網站服務的對象所在地，但以台灣來說，大品牌的雲端主機服務隨遍佈世界各地，如[Amazon Web Services(AWS)](https://aws.amazon.com/tw)、[Microsoft Azure](https://azure.microsoft.com/zh-tw/)、[Google Cloud Platform(GCP)](https://cloud.google.com/)、[Linode](https://www.linode.com/)等，但大部分在台灣都沒有架設機房，只有Google在台灣的彰化地區有架設機房，但都有在亞洲其他國家架設機房，鄰近台灣最近的日本也一定佈署機房，因此除了可以考慮使用Google雲端服務外，亦可以考慮使用其他鄰近國家機房位置，例如日本、新加坡，經實測日本機房連線至台灣，連線速度上都不會太差，至少在使用者體驗上不會有明顯的差異。

### 上述主機服務不適用中國地區

若服務的客戶為中國地區，上述的主機都不能直接使用。因為在中國大陸，所有網站都必須ICP備案，由當地政府監管。而ICP備案，必須有實際在中國大陸登記公司行號才能申請。不申請ICP備案並非你的網站就一定在中國地區觀看，只是在多數地區非常可能會被封鎖或屏遮，唯有申請ICP備案後，才能確保網站能正常顯示在中國地區。此外，即使申請ICP備案，主機都建議使用中國當地認可的主機商，如阿里雲](https://www.vultr.com/vultr-vs-digitalocean)、或者[騰訊云](https://cloud.tencent.com/product/cvm)主機服務，並免外地主機服務商IP位置被中國政府封鎖，導致網站無法顯示。
